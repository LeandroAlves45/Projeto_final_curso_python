"# Projeto_final_curso_python" 
Car Rental Management Web Application

Overview

This project is a web-based Car Rental Management System developed in Python.
It allows users to manage car rentals, track availability, handle bookings, and visualize key business indicators through a dashboard.

Features
- User authentication with session management
- Car catalog management
- Rental creation with date validation
- Automatic availability control
- Dashboard with indicators and charts
- Data export functionality

Technology Stack
- Python 3
- Flask
- SQLite
- Jinja2
- HTML / CSS

Project Structure
projeto_web/
- project_web.py
- database/
- templates/
- static/
- exports/

Installation
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies

Running the Application
python project_web.py

Access via http://127.0.0.1:5000

Dashboard
Provides rental statistics, availability indicators, and performance metrics.

Security Notes
Session-based authentication is used.
Debug mode should be disabled in production.
